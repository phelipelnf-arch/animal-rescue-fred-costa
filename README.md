# 🐾 Animal Rescue: Fred Costa

Um jogo de resgate de animais em 3D desenvolvido em Unity para plataformas Mobile e Web. O jogador controla o Fred Costa em uma missão para resgatar cães e gatos feridos, maltratados e abandonados, superando diversos obstáculos.

## 📋 Visão Geral do Projeto

- **Plataforma**: Mobile (iOS/Android) e Web (WebGL)
- **Engine**: Unity 2022 LTS ou superior
- **Linguagem**: C#
- **Estilo Visual**: 3D
- **Personagem Principal**: Fred Costa

## 🎮 Mecânicas Principais

1. **Resgate de Animais**: Encontre e resgate cães e gatos feridos
2. **Obstáculos**: Supere desafios ambientais e inimigos
3. **Progressão**: Níveis com dificuldade crescente
4. **Cuidado**: Trate animais feridos antes de salvá-los
5. **Inventário**: Sistema de itens para resgate e cura

## 🚀 Como Começar

### Pré-requisitos
- Unity 2022 LTS ou superior
- Visual Studio Code ou Visual Studio
- Git

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/phelipelnf-arch/animal-rescue-fred-costa.git
cd animal-rescue-fred-costa
```

2. Abra o projeto em Unity

3. Navegue até a cena principal: `Assets/Scenes/MainMenu.unity`
